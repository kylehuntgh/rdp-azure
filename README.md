<h1>Remote Desktop and Azure - How To</h1>
This guide will demonstrate how to connect to a Azure virtual machine with remote desktop.<br />


<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop

<h2>Operating Systems Used </h2>

- Windows 10</b> (21H2)

<h2>Prerequisites</h2>

- Create Azure Account (free 200$ credit)
- Create Virtual Machine (to connect to through RDP)
